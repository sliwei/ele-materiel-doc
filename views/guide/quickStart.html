# 安装

# 安装依赖

npm i @msb/ele-materiel -D --registry http://fusion.meishubao.com:9001

TIP

插件放在私有库中，建议使用nrm保存美术宝前端私有库

nrm add http://fusion.meishubao.com:9001 meishubao
nrm use meishubao

# 完整引入

import Vue from 'vue';
import ElementUI from 'element-ui';
import 'element-ui/lib/theme-chalk/index.css';
import EleMateriel from '@msb/ele-materiel';
import '@msb/ele-materiel/lib/css/ele-materiel.css'

import App from './App.vue';

Vue.use(ElementUI);
Vue.use(EleMateriel);

new Vue({
  el: '#app',
  render: h => h(App)
});

TIP

ele-materiel依赖element-ui，所以需要安装element-ui
